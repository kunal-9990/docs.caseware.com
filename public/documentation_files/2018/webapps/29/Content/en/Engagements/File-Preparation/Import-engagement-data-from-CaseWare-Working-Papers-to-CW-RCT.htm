Add the CloudBridge document to Working Papers

Before you can export data from a Working Papers engagement, you will need to add the CloudBridge (CLBG) document to the Document Manager.

To add the CLBG document to Working Papers:

  1. Ensure your copy of Working Papers is integrated with your Cloud firm. To learn more, see Integrate Working Papers with Cloud.

  2. Ensure that you have installed CloudBridge. To download CloudBridge, see Import engagement data from CaseWare Working Papers to CaseWare ReviewCompTax.

  3. Open your client file in Working Papers.

  4. From the Document tab, click From Library | CloudBridge. The Document Library dialog displays.

  5. Click CLBG and click Add.

The CLBG document displays in the Document Manager. You can now use the CLBG document to export your engagement data.

Transfer engagement data from Working Papers

After you add CLBG document to a Working Papers engagement, you can transfer your trial balance and general ledger tax data to CaseWare ReviewCompTax.

To export your engagement data to CaseWare ReviewCompTax using CloudBridge:

  1. In the Document Manager, double-click CLBG. The CLBG document displays.

  2. In the Step 1 section, choose an appropriate entity and engagement.

  3. In the Step 2 section, choose the appropriate trial balance data to export.

  4. In the Step 2 section, select the Use remap file check box.

    The remap file is used to transfer the current account mapping structure in Working Papers to CaseWare RCT. If your engagement uses the default Working Papers mapping or was created with the PEG template, use the remap file located in the folder where you installed CloudBridge (C:\Program Files (x86)\CaseWare\Scripts\cloudbridge).

    Note: If your engagement uses a custom mapping structure specific to your firm, create a custom remap file then add it.

  5. Select Choose file to browse and add the remap file.

  6. (Optional) If you have other journal entries related to donations, meals and entertainment expenses or foreign advertising, the Step 3 section is available in the page. Step 3 section allows you to transfer GL data to the appropriate Tax Worksheet in the CaseWare RCT engagement.

    Note: Step 3 section is only available on the page if your engagement file contains other entries.

    In this example, we’ll add gifts to client to the Donations worksheet.

  7. Once you're done, select Transfer Data.

Your exported engagement data is available in CaseWare ReviewCompTax. If you exported general ledger data, you can see all of your updated tax sheets by selecting Documents | More Actions ( ) | Hidden Items in your CaseWare ReviewCompTax engagement. Some of your tax sheets may be visible by default.

Create a custom remap file (.ini)

If your firm uses a custom mapping structure, you need to create your own remap file (.ini) file based on CaseWare RCT mapping structure.

To create a custom remap file:

  1. Copy components.

    1. Open your Working Papers file.

      2. We recommend that you create a backup copy of your file before you proceed (File | Back Up).

    2. On the ribbon, click File | Copy Components.

      3. The Copy Components Wizard displays.

    3. Click Copy Into This File.

    4. On the Select Source page, click the File tab, then click Browse...

    5. Navigate to C:\Program Files (x86)\CaseWare\Scripts\cloudbridge\CaseWare RCT and select CaseWare RCT.ac, then click Next.

    6. On the Copy Components page, select Groupings/Mapping.

    7. Select Mapping and click Options…

      8. The Mapping Copy Options dialog displays.

  2. Remap numbers.

    1. Click Remap.

      The Remap dialog displays.

    2. Drag a mapping number from the Destination File Mapping Numbers list and drop it into the appropriate mapping number in the Source File Mapping Numbers list.

    3. Repeat the process to assign additional mapping numbers.

    4. Once you're done, click Apply to save your settings, then click OK.

  3. Generate the .ini file.

    1. In the Mapping Copy Options dialog, click Cancel.

    2. In the Copy Components Wizard, click Cancel.

      3. A remap file with the name copy.ini is created in the folder of your Working Papers file ( C:\Program Files (x86)\CaseWare\Data).

You can now use this remap file in Cloudbridge to

transfer your Working Papers data to CaseWare RCT.
