Set up the Risk of Material Misstatements (RMM) Assessment report

You can enable the RMM Assessment report to allow users to assess RMM per financial area. For example, they’d review Cash and decide whether or not there is a risk that cash is materially misstated.

To enable the Risk Assessment report in your product, add the property for it in the product.json file. See product features for more information.

Once you enable the report in your product, you can add it in your draft and set it up.

To add the risk assessment report, select Add () at the documents page, then choose Risk Assessment Report.

To set up the Risk Assessment report:

  1. At the Filters drop-down list on the top left, select the filter you want to apply to the document.

  2. At Settings (), modify the default settings if needed.

    3. The following options are available:

Setting

Description

Whether or not to include the Conclusion section at the end of the report

Additional column

Whether or not to include an extra column next to Reason column for each financial area

Allow assessments under Cycles or Areas

Whether or not to allow assessments under cycles or areas.

    2. If you want the information grouped, you select to group by Areas or Cycles.

Add or remove a link in the guidance information

You can add links to a URL, an email or a phone number.

To add a link:

Select Add link () and enter the following information in the Link Info tab:

  • Display Text - the selected text is the default Display text for the link. If you haven’t selected text, enter the Display Text in this field.

  • Link Type - select a type from the drop-down list. Based on your selection, the dialog displays different sets of fields that you need to specify:

    Link type

    Fields

    URL

    Protocol - select the link protocol (http, https, ftp, news or other)

    URL - enter the URL

    Link to anchor in the text

    This feature is currently not functional

    Email

    E-mail address - enter the email address

    Message Subject - enter a subject for the email message

    Message Body - enter the email message body

    Phone

    Phone number - enter the phone number

In the Target tab, select where you want the link target to open. The default is in a new browser window.

You can also enter more information in the Advanced tab. Note that if you add a link to a file download, you can select the Force Download check box in this tab so the file download starts once the user selects the link.

Once you’re done, select OK.

To remove a link in a text section, select the link text then select Remove Link ().
