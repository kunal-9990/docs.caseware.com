Formatting client data from desktop software for importing into CaseWare Cloud

The tool and steps required for importing client data from your computer into CaseWare Cloud differs depending on what software package you're importing from.

Importing from CaseWare Working Papers

To import client data from a Working Papers file, you'll need to use the CaseWare Cloud Import Utility to format the data into a transformed .zip file format.

CaseWare Cloud Import Utility - Beta (.exe)

 

To format client data from a Working Papers file using the Cloud Import Utility:

  1. Download the latest version of the utility.

  2. Open the Utility. Select Existing File.

  3. Browse for the file path of the client file whose data you want to import.

  4. Click Next. You will see a prompt with the output folder you'll need to open to access your new Transformer file. Select Finish to perform the conversion.

 

The output file can be uploaded into a variety of CaseWare Cloud products.

CaseWare Cloud Import Utility - Beta (.exe)

To format client data using the Cloud Import Utility:

  1. Download the latest version of the utility.

  2. Open the Utility. Select Other Data Source.

  3. In the Import From and Version fields, specify the accounting software you're exporting from.

  4. Browse for the Import Path of the client file you want to import. If multiple client files are located in this folder, you will need to specify which file you want to import in the Select Client ID File field.

  5. Select Next. You will see a prompt with the output folder you'll need to open to access your new Transformer file. Select Finish to perform the conversion.

The output file can be uploaded into a variety of CaseWare Cloud products.
