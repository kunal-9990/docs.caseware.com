Performing a tax engagement

Canadian CRT enables you to perform the required tasks of a tax engagement along with the Compilation and Review functionality.

  • complete the Tax Optimizer checklist

  • upload your client's tax data

  • complete the Tax Preparer profile

  • complete the provincial and federal forms using the Tax worksheets

  • run a diagnostic check to ensure there are no outstanding errors

  • file the return electronically via EFILE or create a PDF for your client if required

Setting up the tax engagement

As with the Compilation and Review engagements, a tax engagement starts with an Optimizer checklist - in this case, the Tax Optimizer.

Note: This checklist will only display when you have specified in the first Optimizer that you will be performing a tax engagement.

Complete these procedures as follows:

Procedure Response
Is the corporation filing an amended tax return? Yes
Is this corporation a member of a partnership? No
Is this corporation part of an association or related group? No
Does amalgamation / dissolution / year-end close, etc. apply? No
Is the corporation a resident of Canada? Yes
Province of jurisdiction of the corporation Ontario

Complete the document with the details of the entity you're performing the tax engagement for, as well as the characteristics of the engagement.

The engagement will automatically populate with the appropriate provincial and federal tax forms, as well as Review and Compilation – Canada's worksheets. These documents will appear in the Document Map on the left panel. You can choose to complete the new documents as they appear with each answered procedure, or finish the checklist and complete the added documents later.

Import tax data

Cdn CorpRevTax gives you a few different ways to import the engagement's tax data, Open any document in the Tax section.

Select Import to view the import options.

Note: If you have already imported a trial balance and assigned groupings for a Compilation or Review engagement, the same values will automatically update the tax engagement's GIFI forms.

 

GIFI import

Select this option to browse your computer for a GIFI output file from your desktop accounting package.

TaxPrep CSV Import

Select this option to upload a .csv output file generated from TaxPrep v2017 v1.1.

To export a .csv file from TaxPrep:

  1. Open the client file in TaxPrep.
  2. Perform a Roll Forward.
  3. Select File | Export Tax Return....
  4. Make sure the File Type is set to .csv, then select OK.
  5. Enter a file name and path, then select Save.

T2 Auto-fill

If you've registered for the CRA's T2 Auto-fill program, you can import the engagement's tax data automatically based on the client's previously filed returns.

To transfer tax engagement data from the CRA:

  1. In any open Tax document, select Import | T2 Auto-fill.

  2. Select Yes in the following prompt.

  3. The CRA website opens in a new browser tab. Log into your CRA account using your credentials. On the request confirmation page, select Next.

  4. The data from CRA is now linked. select the data types you want to import into your engagement, then select Download.

  5. Before you can complete the import, you will be asked to compare the values from the CRA against the existing values that have already been entered for the fields, if any. The CRA values are displayed in the left column, while the engagement's current values are displayed on the right. For each line, select the checkbox on the right to use the CRA value. To use the CRA values for all lines, select the large checkbox at the top of the modal. When you're done, select Transfer.

  6. The values are transferred to your engagement and a confirmation modal is displayed. Select 5 Year History to review the historical data, or select Close.

 

Download TaxBridge

TaxBridge is an add-on for CaseWare Working Papers that enables you to link a Working Papers engagement file to a Cloud entity in order to import your engagement's tax data into your Cloud apps. You should use this option if you already have a Working Papers engagement file that contains GIFI data.

Select Download TaxBridge from the Import menu to download the latest version of TaxBridge.

To create a link for the TaxBridge:

  1. Open and run the TaxBridge install package you downloaded from CaseWare Cloud.

  2. Open your client file in CaseWare Working Papers.

  3. In the Document Manager, right click and select New | Document Link.

  4. In the New Document Link dialog, enter TaxBridge as the Name. In the File/URL field enter the following address: cw:program/scripts/taxbridge/index.html.

  5. Click OK to add the document link.

  6. Double-click to open the link.

With the link created, select TaxBridge from the Document Manager. This will open TaxBridge in your Working Papers browser. Follow the instructions in TaxBridge to transfer the data.

Complete the Tax Preparer profile document

The TPP - Tax Preparer Profile document requires you to enter key information about the entity performing the tax engagement.

Once this document is completed and signed off on, you're ready to begin your tax engagement work.

Complete the tax documents

With the tax data imported and the profile documents completed, you're ready to begin completing the engagement's federal and provincial tax forms. Since you completed the Tax Optimizer document, Review and Compilation – Canada should only list the forms that are relevant to your tax engagement.

To see the complete list of available tax forms, not just the ones that Review and Compilation – Canada has determined are relevant to your engagement, select the Edit icon () on the Documents page, or open the Tax Forms List document. Use your browser's search function (ctrl-f) to help locate specific documents.

Your selections also determine which workcharts, if any, are listed. Workcharts are documents designed to streamline your engagement work.

Navigate between related tax documents

Review and Compilation – Canada makes it easy to navigate between workcharts and their linked schedule documents. The CompTax interface offers two options for moving between these documents.

Jump To Source

When you select a field that has a single source, you will see this dialog:

When you select Jump To Source, the source document opens and jumps to the appropriate line, with the exact source field highlighted in flashing blue.

Viewer

In documents where an entire row of data is derived from one specific document, you’ll see a Viewer icon () to the right of the row. This feature opens the linked document in a pop-up display without opening a new browser window. For example, each class listed in the S8 - Capital Class Allocation document is linked to its corresponding page from the CCA Worksheet.

Example workflow

In this example, your client has purchased a new fixed asset, in this case, a newly purchased building. The S8 - CCA Workchart is used to manage fixed assets, so this is the workchart you would select.

With the workchart open, you will want to select the appropriate class for the firm's new fixed asset. Select the add icon () in the CCA Class field.

Select the appropriate CCA class, then select OK.

Scroll to the bottom of the workchart, then select add (). On the new line, select the Viewer icon (). A new modal opens, and you can enter the new asset's information.

Correct outstanding errors or issues

If there are any outstanding issues with your return, you will be unable to file it. Select the DIAG - Diagnostics Workchart document to view the engagement's outstanding issues. Select E-File Diagnostics to filter the list of issues to only display those that will prevent an electronic filing.

To view and fix an issue, select it, then select Jump To.

E-filing the return to the CRA

With the tax data imported, the engagement work complete, and all outstanding issues fixed, you can begin to plan your electronic filing via the CRA's EFILE program. Select EFILE from the top menu.

Note: Review and Compilation – Canada currently does not enable you to submit an RC4649 - Country by Country Return form.

From the EI - Efile Information document, you can file your return to the CRA via one of two sets of credentials - if your firm is a Certified Tax Preparer, you can enter your CRA Login Information; alternatively, you can enter a WACode (Web Access Code) provided by the client.

If you require a Web Access Code and have not yet been provided with one, select WAC - Web Access Code Online Request and complete the request form.

Printing the return

If your client requests a copy of the return for their records, you can create a PDF file which can then be saved to disk and shared with the client electronically

Select Print from any open tax document to select the document you want to print.
